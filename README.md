# 🚀 Apex CRM Dashboard

This project is built using modern frontend technologies for fast performance and clean UI.

---

## 🛠️ Tech Stack

* ⚛️ React
* ⚡ Vite
* 🎨 Tailwind CSS

---

## 📦 Installation

> ⚠️ Note: `node_modules` folder is not included in this repository.

To install dependencies, run:
npm install



## 🎨 Styling (Tailwind CSS)

* Tailwind CSS is used for styling the UI.
* Custom utility classes for **font sizes** are defined in:

tailwind.config.js
These custom font utilities help maintain consistent typography across the project.
